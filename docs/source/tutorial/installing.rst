Installing VSMlib
=================

.. currentmodule:: vsmlib


System requirements
-------------------

- Python 3.5 or later

Method 1: Pip-install
--------------------

The latest stable version:

```python
pip install vsmlib
```
The latest development version:

```python
pip3 install git+https://github.com/undertherain/vsmlib.git 
```

Method 2: Clone or download the github repo
------------------------------------------

You can avoid intalling vsmlib system-wide. Simply download and unpack the github repo into your project's working directory. 

Either way, you can access the vsmlib's modules by issuing

```python
import vsmlib
```

at the beginning of your code.
